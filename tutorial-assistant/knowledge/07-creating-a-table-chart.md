A table chart shows data as rows and columns, either aggregated or as raw records.

To create a table chart:

1. Open **Charts** and click **+ Chart**.
2. Choose a dataset and select **Table** from the chart-type gallery, then click **Create new chart**.
3. In Explore's **Data** tab, pick a **Query mode**:
   - **Aggregate** groups rows. Add grouping columns to **Dimensions** and aggregations to **Metrics**.
   - **Raw records** lists rows as they are. Add the columns to show to **Columns**.
4. Use **Filters** to restrict rows and **Row Limit** to cap how many are returned.
5. Click **Create chart** to render, then **Save** to store the chart.

Table-specific options live in the **Customize** tab, including **Search box**, cell bars for numeric columns, and column alignment. Sorting is available by clicking a column header in the rendered table.

Tables are a good first chart for checking that a dataset returns the values you expect before building more visual chart types from it.
